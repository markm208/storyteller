Storyteller: Docs

Storyteller is a programming language independent editor plugin that can generate web pages that recreate entire programming sessions in a character by character animation. Developers can then playback their work and add comments to their code at different points in the evolution of the code. Playbacks are simple web pages that can easily be shared with others.

The author of a playback can use it to reflect on their work to understand what went well and what didn't and then create a narrative about how and why the code evolved the way it did. Collections of these playbacks can be used as a shared learning resource for members of a team, a way to give a presentation with code as the main focus, as version control check-in documentation, or as a personal journal.
